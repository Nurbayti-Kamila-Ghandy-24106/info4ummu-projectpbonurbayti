# info4ummu-projectpbonurbayti
Sintaks Inheritance dalam Python-Pewarisan Atribut dan Method

# Deskripsi
Projek ini dibuat untuk memenuhi tugas mata kuliah
Pemrograman Berorientasi Objek (PBO) kelas info 4
Program ini menggunakan konsep Object Oriented Programming (OOP) seperti :
- class
- object
- Method
- Pewarisan (inheritance)
Program bertujuan unruk memberikan contoh penerapan OOP sederhana Mmenggunakan bahasa pemograman Python

# Fitur
-menampilkan aktivitas hewan (makan,tidur,minum)
-menampilkan suara hewan(kucing)
-menampilkan informasi kendaraan
-simulasi pembelian dan aktivitas motor

# Struktur
main.py : Program Utama
hewan.py :Class Hewan dan Kucing
kendaraan.py : Class Kendaraan dan Motor

# Cara Menjalankan
- Pastikan Python sudah terinstal di komputer
- Clone atau download repository
- Masuk ke direktori project
- Jalankan perintah

Bast
python main.py
